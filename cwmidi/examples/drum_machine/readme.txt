This example atempts to simulate a drum machine.
It plays drums randomly. 
To build the example program type :

make

from a command shell.
A test program 'prog.exe' will be in the bin folder.
It's a command line application. to start it type:

./run.sh

if you are using the MinGW Shell

or just:

run

if you are using the cmd command line shell for windows.
